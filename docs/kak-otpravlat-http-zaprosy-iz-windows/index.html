Как отправлять HTTP-запросы в Windows

October 31, 2020.
Время чтения — 2 мин.
#http #windows #curl #scoop

При разработке программного обеспечения и при мониторинге серверов порой необходимо отправлять HTTP-запросы. Самые популярные из них это: POST и GET.

В мире Linux/Unix/MacOS чаще всего используют утилиту curl, которая устанавливается одной командой. В Windows вы также можете установить curl.

Способ №1 — Установка через менеджер пакетов

Самый простой и быстрый вариант — установить через менеджер пакетов scoop:

scoop install curl

Способ №2 — Установка с официального сайта

  1. Скачиваем архив с официального сайта https://curl.haxx.se/windows/ архив.

  2. Распаковываем куда-нибудь. Например, c:\apps\curl. Содержимое каталога должно выглядеть подобным образом:

    5. Далее нажимаем кнопки ОК пока не закроются все окна.
